# Dataset Card: wasmagent-smoke-traces-v1

## Dataset Summary
A small, fully sanitized smoke dataset for the WasmAgent evidence pipeline. Contains synthetic AEP records and rollout traces for pipeline testing and onboarding.

## Dataset Details
- **Version**: 0.1.0
- **Created**: 2026-06-26
- **Records**: 5 AEP records, 3 rollout records
- **Schema versions**: aep/v0.1, rollout-wire/v1
- **Source**: Synthetic generation (no real user data)

## Data Collection
All records are fully synthetic — generated by scripts/generate-smoke-data.py. No real agent runs, no real user tasks, no real API keys.

## Contamination Check
- Algorithm: 8-gram Jaccard similarity
- Threshold: 0.8
- Flagged records: 0/8
- Reference corpora: GSM8K, HumanEval, MBPP

## Schema
- AEP records: see wasmagent-js/packages/aep/src/types.ts (aep/v0.1)
- Rollout records: see wasmagent-js/packages/core/src/ranking/schemas/ (rollout-wire/v1)

## Known Limitations
- Synthetic data only — not representative of real agent behavior
- Small scale — not suitable for training, only for pipeline testing
