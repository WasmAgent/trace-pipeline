# Dataset Card: wasmagent-smoke-traces-v1

## Dataset Summary

A small, fully sanitized smoke dataset for the WasmAgent evidence pipeline. Contains synthetic AEP records and rollout traces for pipeline testing and onboarding.

## Dataset Details

- **Version**: 0.1.0
- **Created**: 2026-06-26
- **Records**: 5 AEP records (`aep-smoke.jsonl`), 3 rollout records (`rollout-smoke.jsonl`)
- **Schema versions**: aep/v0.1, rollout-wire/v1
- **Source**: Synthetic generation via bscode reference deployment (no real user data)

## Data Collection

All records are fully synthetic — generated by `scripts/generate-smoke-data.py` using the bscode reference deployment. No real agent runs, no real user tasks, no real API keys.

**Provenance**: Records originate from bscode's synthetic task harness, exported via `AEPEmitter` in `@wasmagent/aep`, then ingested and validated through the evomerge pipeline before inclusion.

## AEP Evidence Quality

| File | Records | Schema | Model providers |
|---|---|---|---|
| `aep-smoke.jsonl` | 5 | aep/v0.1 | anthropic |
| `rollout-smoke.jsonl` | 3 | rollout-wire/v1 | — |

**Admission criteria**: Evidence Admission Score >= 0.6 (all 5 AEP records pass).

## Rollout Wire Stats

- `rollout-smoke.jsonl`: 3 records, schema rollout-wire/v1
- Each record contains a `tool_call_sequence`, `branch_index`, `temperature`, and `session_id`
- Records cover 3 distinct task branches across 1 session

## Contamination Check

- Algorithm: 8-gram Jaccard similarity
- Threshold: 0.8
- Flagged records: 0/8
- Reference corpora: GSM8K, HumanEval, MBPP
- Validation tool: `evomerge validate-aep` (all records confirmed clean)

## Schema

- AEP records: see `wasmagent-js/packages/aep/src/types.ts` (aep/v0.1)
- Rollout records: see `wasmagent-js/packages/core/src/ranking/schemas/` (rollout-wire/v1)

## Usage

```bash
# Validate AEP records against schema and admission criteria
python -m evomerge validate-aep --input data/smoke/aep-smoke.jsonl

# Generate a full audit report
python -m evomerge audit-report --aep data/smoke/aep-smoke.jsonl
```

## Known Limitations

- Synthetic data only — not representative of real agent behavior
- Small scale — not suitable for training, only for pipeline testing
