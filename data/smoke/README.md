# wasmagent-smoke-traces-v1

A small, fully sanitized, reproducible smoke dataset for the WasmAgent evidence pipeline.

## Contents
- aep-smoke.jsonl — 5 synthetic AEP records (aep/v0.1 schema)
- rollout-smoke.jsonl — 3 synthetic rollout-wire/v1 records

## Purpose
- Verify evomerge validate-aep works on valid input
- Smoke-test the training data export pipeline
- Provide a reproducible fixture for CI and new contributor onboarding

## Generation
Generated by: python scripts/generate-smoke-data.py
Schema version: aep/v0.1 / rollout-wire/v1
Contamination check: clean (8-gram Jaccard against public benchmarks)
